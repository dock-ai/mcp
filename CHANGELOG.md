# Changelog

All notable changes to the Dock AI MCP server will be documented in this file.

## [1.0.0] - 2025-01-05

### Added
- Initial GA release
- `resolve_domain` tool for discovering MCP connectors by business domain
- Multi-location entity support
- Verification levels for trust (0, 1, 2)
- Installation instructions for Claude, ChatGPT, and Mistral Le Chat
- Static resources: Getting Started Guide, Supported Providers
- Prompts: Discover Business, Setup Connector
- Published to official MCP Registry as `co.dockai/mcp`

### Technical
- Streamable HTTP transport
- Hosted version at `https://mcp.dockai.co/mcp`
- Self-hosting support via `uvx dock-ai-mcp`
